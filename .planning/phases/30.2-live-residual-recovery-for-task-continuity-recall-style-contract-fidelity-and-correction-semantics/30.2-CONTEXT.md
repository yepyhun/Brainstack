# Phase 30.2 Context

## Why this phase exists

Live Discord UAT after `v1.0.4` exposed three concrete residual classes that the current roadmap did not yet lock as separate truths:

- task continuity recall:
  - the user gave tasks
  - a `tier2_summary` was actually written
  - but reset-era Hungarian recall still failed
- style-contract fidelity:
  - exact recall worked better
  - but the live canonical scoped `style_contract` row was still `tier2_llm`-sourced and lossy
- same-session correction semantics:
  - the user corrected an em-dash violation in ordinary chat
  - but the system had no clearly defined path from that correction to a stronger active policy state

There is also a fourth, closely coupled truth problem:

- projection truthfulness:
  - compile coverage can currently overstate what the ordinary-turn prompt actually enforces
  - a rule may be compiled and reported active while still being omitted from the bounded ordinary projection

## Inputs

- live DB evidence showed:
  - task summaries existed in continuity
  - recent continuity is session-scoped
  - cross-session continuity search is lexical
  - English task summaries can miss Hungarian task queries
- live DB evidence also showed:
  - the scoped `style_contract` row could still be sourced from `tier2_llm`
  - the stored raw rule text itself was already mutated
- code inspection showed:
  - Brainstack currently says tasks/reminders remain with native owners
  - ordinary chat correction is not automatically the same as explicit behavior-policy correction
  - compiled behavior-policy coverage and ordinary-turn projection semantics are not yet the same contract
  - plain-chat structured teaching is not yet clearly a pre-answer activation path

## Non-goals

- no immediate redesign of task ownership
- no broad rework of second-brain or KG architecture
- no code fix in this planning step
- no acceptance of vague “the model just drifted” explanations without storage and retrieval evidence
- no collapsing of host/runtime enforcement and memory-layer truth into one mixed phase

`xhigh`
