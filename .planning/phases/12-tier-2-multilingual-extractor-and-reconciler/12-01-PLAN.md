## Objective
Implement the real Tier-2 multilingual extraction path and deterministic reconciler on top of the Phase 11 ingest seam, while preserving the non-blocking `sync_turn()` contract.

## Deliverables
1. `tier2_extractor.py`
- bounded transcript-batch prompt assembly
- Hermes auxiliary-client based LLM call path
- strict timeout
- structured candidate parsing
- fake extractor injection path for tests

2. `reconciler.py`
- deterministic application of Tier-2 candidates
- `ADD / UPDATE / NONE / CONFLICT` outcomes
- profile, graph, and bounded continuity writes

3. provider background wiring
- one daemon worker path
- no dropped follow-up work while worker is already active
- session-end wait + final flush behavior
- no second runtime or queue service

4. targeted verification
- non-blocking `sync_turn()` proof
- no dropped batch proof
- session-end flush proof
- installed-runtime proof after reinstall into Bestie

## Non-Negotiables
- `sync_turn()` must not wait for the LLM roundtrip
- no language-specific regex expansion as the Phase 12 answer
- no parallel hidden write path outside the Brainstack store API
- no corpus expansion in this phase
- no new model-facing tool surface

## Verification
- source tests prove scheduling, extraction, and reconciliation behavior
- installed-runtime smoke proves the background Tier-2 path writes to the expected shelves
- GSD summary and state are updated only after both code and runtime proofs pass
