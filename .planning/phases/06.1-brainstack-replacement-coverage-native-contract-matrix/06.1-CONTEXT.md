# Phase 06.1 Context

## Goal
Prove that Brainstack covers every native Hermes memory responsibility that the host actually uses before any final "native memory retired" claim is made.

## Why This Phase Exists
- Phase 6 proved clean host displacement and fail-closed native behavior.
- Phase 6 did not, by itself, prove that Brainstack is already equal-or-better across every host-used native responsibility.
- The user explicitly requires a replacement coverage / native contract matrix before accepting permanent native retirement.

## Contract Scope
- prompt memory replacement
- user/profile replacement
- tool surface ownership
- direct memory-tool fail-closed behavior
- prefetch / recall path
- post-turn sync path
- pre-compress path
- session-end extraction path
- background review / flush replacement or intentional removal

## Required Standard
Every in-scope responsibility must end in exactly one of these states:
- matched by Brainstack
- intentionally superseded by a better Brainstack path
- intentionally dropped, with explicit rationale and no hidden runtime dependence

Anything else is a blocking gap.

## Non-Goals
- Do not benchmark-game or question-shape tune.
- Do not re-enable the native Hermes memory path as a safety blanket.
- Do not claim universal superiority in every future scenario.
- Do prove host-contract replacement for the responsibilities Hermes actually depends on.
