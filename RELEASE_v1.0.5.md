## Brainstack 1.0.5

This release shifts Brainstack back toward a cleaner second-brain role while strengthening the operating substrate behind it.

Highlights:

- hardens personal truth continuity across workspace and identity drift
- upgrades behavior contracts to safer canonical patching and narrower ordinary-turn enforcement
- de-escalates always-on behavior steering so the LLM keeps local turn composition
- adds first-class operating truth records for active work, decisions, commitments, next steps, and external owner pointers
- moves retrieval closer to owner-backed operating truth instead of over-relying on shallow routing cues
- updates the installer and doctor so the shipped Hermes payload stays aligned with the new operating-truth surface

This release is focused on second-brain quality, not benchmark tricks: better continuity, clearer owner boundaries, and less hot-path harnessing.

If you hit an issue or want to improve something, feel free to open a ticket or PR.
